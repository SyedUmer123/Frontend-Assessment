# Frontend Assessment — Tip Calculator

Single-screen tip calculator and bill splitter. Totals update live as you type.

Live demo: https://frontend-assessment-kqctkmr7s-syed-umer-razis-projects.vercel.app

## Run locally

Requirements: Python 3, any modern browser.

```bash
git clone https://github.com/SyedUmer123/Frontend-Assessment.git
cd Frontend-Assessment
python -m http.server 9090
# Open http://localhost:9090
```
